# HDB Price Predictor

## Overview
This applicatiom predicts the selling prices of [HDB apartments](https://www.hdb.gov.sg/cs/infoweb/about-us) in Singapore. Using publicly available sales data, this application builds a machine learning model that takes into account features such as location, lease start date, floor level, as well as selling date to predict the most likely price a particular apartment will be selling for.
